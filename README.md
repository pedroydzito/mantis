# Mantis

Editor de vídeo com IA para Windows.

## Baixar e usar

1. Vá em **[Releases](https://github.com/pedroydzito/mantis/releases/latest)** e baixe **Mantis-win.zip**.
2. **Importante (1ª vez):** clique com o botão direito no **Mantis-win.zip** → **Propriedades** → marque **"Desbloquear"** → **OK**. Isso evita o aviso do Windows SmartScreen.
3. Extraia a pasta e rode **Mantis.exe**. (Se ainda assim aparecer "O Windows protegeu o computador", clique em **Mais informações → Executar assim mesmo** — só na primeira vez.)
4. Ao abrir, se houver uma versão nova, o app avisa e atualiza sozinho.

Veja o que mudou em cada versão no [CHANGELOG](CHANGELOG.md).
