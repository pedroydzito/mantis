# Mantis

Editor de vídeo com IA para Windows.

## Baixar e usar
1. Vá em **[Releases](https://github.com/pedroydzito/mantis/releases/latest)** e baixe **Mantis-win.zip**.
2. Extraia a pasta e rode **Mantis.exe**.
3. Ao abrir, se houver versão nova, o app avisa e atualiza sozinho.

Veja as mudanças no [CHANGELOG](CHANGELOG.md).
